## 🏁 Quick Start

### 1️⃣ Get Started (2 minutes)

```bash
# Use this template on GitHub, then:
git clone https://github.com/yourusername/your-project.git
cd your-project
npm install
```

### 2️⃣ Automated Setup

```bash
npm run setup:project
```

This interactive script will:

- Guide you through project configuration
- Set up your business information
- Configure deployment settings
- Generate initial documentation

### 3️⃣ Start Development

```bash
npm run dev
```

Open http://localhost:5173 to see your site!

### 4️⃣ Deploy to Production

```bash
npm run build
git add .
git commit -m "feat: initial deployment"
git push origin main
```

Your site will automatically deploy to GitHub Pages! 🎉

## 🛠️ Available Scripts

### Development

```bash
npm run dev          # Start development server with HMR
npm run dev:host     # Start dev server accessible on network
npm run preview      # Preview production build locally
```

### Code Quality

```bash
npm run validate     # Run all quality checks
npm run lint         # Check JavaScript with ESLint
npm run lint:css     # Check CSS with Stylelint
npm run lint:html    # Check HTML with HTMLHint
npm run format       # Format code with Prettier
npm run format:check # Check if code is properly formatted
```

### Testing

```bash
npm test             # Run unit tests
npm run test:watch   # Run tests in watch mode
npm run test:ui      # Run tests with UI
npm run test:coverage # Generate coverage report
```

### Build & Deploy

```bash
npm run build        # Build for production
npm run deploy       # Deploy to GitHub Pages
npm run lighthouse   # Run Lighthouse audit
```

### Project Management

```bash
npm run setup:project    # Interactive project setup
npm run analyze:bundle   # Analyze bundle size
npm run check:updates    # Check for dependency updates
```
